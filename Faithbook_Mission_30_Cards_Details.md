# Faithbook Mission Cards — 30张卡片详情与图像描述

> 用途：给设计、排版、Canva、PPT 或印刷制作时参考。  
> Mission Card 的定位是：**因为神的爱与福音，一起去爱人、服侍人、影响人**。  
> 卡片正面可以显示：标题、任务描述、参与人数、需求、限制、奖励。  
> 建议让 Faith（❤️）需求偏高，因为 Mission 的动机是从信仰出发。

---

## 统一卡片设计建议

- **卡片标题**：MISSION
- **卡片风格**：温暖、行动感、团队合作、Pastel 色系
- **背景**：米白 / 淡绿 / 淡蓝 / 淡黄
- **边框**：圆角卡片框，配小星星、小爱心、小叶子装饰
- **编号**：左上角彩色圆形编号
- **主图位置**：上方或中间
- **任务内容**：简短但有画面感
- **规则区块**：
  - 参与人数
  - 需求
  - 限制（如有）
  - 奖励
- **视觉重点**：❤️ Faith / ⚡ Energy / 🤝 Relationship / 🤍 Mission Token / 💛 Support Token

---

## Mission 基本机制

- Mission 的需求由**参与玩家共同贡献总和**完成。
- 贡献后的属性会被消耗。
- 每张 Mission 有参与人数限制，避免全体玩家用很少资源一起获得奖励。
- 完成 Mission 后，团队获得对应奖励。
- 如果没人能参加或资源不足，Mission 可以进入 **Pending 状态**，放在棋盘中央，下一轮再次尝试。

---

## Icon / Image Direction Legend

| 图像方向 | 象征 |
|---|---|
| 礼物 / 食物 / 关怀包 | 实际关怀、给予 |
| 小组围坐 | 真实分享、关系连接 |
| 祷告手 / 十字架 / 光 | Faith、祷告、属灵动力 |
| 打开门 / 新朋友 | 欢迎、邀请、福音机会 |
| 老人 / 孩子 / 社区 | 教会外的人、被忽略的人 |
| 小灯 / 蜡烛 | 成为别人的光、陪伴、盼望 |
| 卡片 / 纸条 | 鼓励、祝福、被记得 |

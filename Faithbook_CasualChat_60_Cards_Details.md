# Faithbook Casual Chat Cards — 60张卡片详情与图像描述

> 用途：给设计、排版、Canva、PPT 或印刷制作时参考。  
> Casual Chat 的定位是：**轻松连接，不是深层分享**。  
> 卡片问题应该容易回答、生活化、不尴尬，让两位玩家自然开始聊天。

---

## 统一卡片设计建议

- **卡片标题**：CASUAL CHAT
- **卡片风格**：可爱、轻松、温暖、Pastel 色系
- **背景**：米白 / 淡黄 / 淡蓝 / 淡绿 / 淡粉
- **边框**：圆角卡片框，配小星星、小爱心、小云朵装饰
- **编号**：左上角彩色圆形编号
- **主图位置**：卡片中下方或右下角
- **文字排版**：问题居中，字体清楚，留白充足
- **互动感**：看起来像朋友之间轻松聊天的问题卡

---

## Casual Chat 使用机制

1. 玩家进入 Rest 格并选择 **Casual Chat**
2. 先选择一位玩家聊天
3. 抽一张 Casual Chat Card
4. 两位玩家简单聊 30秒–1分钟
5. 双方获得：**🤝 Relationship +1**

---

## Icon / Image Direction Legend

| 图像方向 | 象征 |
|---|---|
| 太阳 / 星星 | 开心、轻松、正面情绪 |
| 饮料 / 食物 | 日常、轻松话题 |
| 音符 / 电影 / 书 | 喜好、兴趣、分享 |
| 日历 / 旅行箱 | 时间安排、期待、旅行 |
| 对话泡泡 | 连接、聊天、被理解 |
| 动物 / Emoji | 有趣、轻松破冰 |
| 小组人物 | 朋友、团契、关系连接 |

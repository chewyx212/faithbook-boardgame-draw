# Faithbook Real Talk Cards — 40张卡片详情与图像描述

> 用途：给设计、排版、Canva、PPT 或印刷制作时参考。  
> 卡片风格建议：温柔、可爱、干净、Pastel 色系；问题文字居中或垂直排版，搭配简单插画，不要让画面太沉重。

---

## 统一卡片设计建议

- **卡片标题**：REAL TALK
- **卡片尺寸**：210mm × 297mm（A4 Portrait）
- **整体风格**：温柔、陪伴感、可爱但不过度幼稚
- **背景**：白色 / 米白色 / 淡粉 / 淡蓝 / 淡绿 / 淡紫
- **边框**：圆角细线边框
- **编号**：左上角彩色圆形编号
- **主图位置**：卡片下方或中下方
- **问题文字**：清楚、温柔、留白足够

---

## Icon Legend

| Icon / 图像方向 | 象征 |
|---|---|
| 爱心 | 被爱、被接住、真实感受 |
| 云 / 雨 | 低潮、压力、疲惫 |
| 信封 | 想说出口但还没说的话 |
| 锁头 | 隐藏的情绪 |
| 背包 | 长期背负的压力 |
| 十字架 / 祷告手 | 与神的对话、信仰、交托 |
| 蜡烛 | 平安、安静、盼望 |
| 对话泡泡 | 分享、回应、被聆听 |

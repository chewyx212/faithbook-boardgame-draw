# Faithbook Life Event Cards — 60张卡片详情与图像描述

> 用途：给设计、排版、Canva、PPT 或印刷制作时参考。  
> 卡片风格建议：可爱、温柔、生活化、Pastel 色系；每张卡有标题、生活情境、A/B/C选择和一张小插画。  
> 注意：卡片正面**不要显示属性增减结果**，属性结果只给主持人参考。

---

## 统一卡片设计建议

- **卡片标题**：LIFE EVENT
- **卡片尺寸建议**：可做成 A4 / A6 / 小卡比例，依打印需求调整
- **整体风格**：青年小组、温柔、真实、可爱但不幼稚
- **背景**：白色 / 米白色 / 淡粉 / 淡蓝 / 淡绿 / 淡紫
- **边框**：圆角细线边框
- **编号**：左上角彩色圆形编号
- **主图位置**：中下方或右下方
- **问题文字**：清楚、简短、有留白
- **选择格式**：A / B / C 三个选项

---

## Icon / Image Direction Legend

| 图像方向 | 象征 |
|---|---|
| 手机 / 对话泡泡 | 讯息、沟通、关系 |
| 云 / 雨 / 低电量 | 疲惫、低潮、压力 |
| 爱心 / 感谢卡 | 被鼓励、被爱、被接住 |
| 门 / 路牌 / 光 | 新机会、方向、未来 |
| 书本 / 电脑 / 待办 | 学业、工作、压力 |
| 小组人物 | 团队、关系、小组氛围 |
| 十字架 / 祷告手 | Faith、Prayer、属灵状态 |
